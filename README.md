## Descrição
Projeto em Java que simula o funcionamento de um sistema de entregas para e-commerce, utilizando conceitos de Programação Orientada a Objetos.

## Funcionalidades
- Cadastro de entregadores
- Criação de pedidos
- Diferentes tipos de entrega
- Atualização de status dos pedidos
- Cálculo de taxa de entrega
- Verificação de quantidade máxima de produtos
- Menu interativo utilizando Scanner

## Estrutura
- **Entregador**: classe abstrata com informações comuns dos entregadores
- **EntregadorCarro**: representa entregas realizadas de carro
- **EntregadorMoto**: representa entregas realizadas de moto
- **EntregadorBike**: representa entregas realizadas de bicicleta
- **Pedido**: representa os pedidos realizados pelos clientes
- **TaxaDeEntrega**: interface responsável pelo cálculo de taxa
- **Main**: executa o sistema e o menu interativo
